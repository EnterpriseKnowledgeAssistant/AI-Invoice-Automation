# AI Invoice Automation

## Features

- Upload Invoice PDF
- Extract text using PDF parser
- AI-powered invoice extraction using Google Gemini
- Manual review of extracted data
- Automatic department routing
- Approval workflow
- SQLite database storage
- Approve / Reject invoices
- Dashboard showing invoice statistics

## Technologies

- Python
- Streamlit
- Google Gemini API
- SQLite
- PyPDF2

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py

## Database

SQLite (invoices.db)

## AI Model

Gemini 2.5 Flash
